# プロジェクトの設定

Pod Installが必要なので、以下の手順でプロジェクトをXcodeで開いてください。<br>

ターミナルで以下のコマンドを実行する
```
git clone https://github.com/LeoAndo/RealmSwiftUISample.git
cd  RealmSwiftUISample
pod install
```
Xcodeでプロジェクトを開くときは、「RealmSwiftUISample.xcworkspace」をダブルクリックして開いてください。<br>
